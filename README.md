# Sistema de Leilão

Este projeto é um sistema de leilão desenvolvido em Java, utilizando NetBeans e Maven.  
Abaixo estão descritos os commits realizados para implementar as principais funcionalidades.

---

## 📌 Histórico de Commits

### Commit #1: Tela CadastroVIEW definida como principal
- Configuração inicial da tela de cadastro como tela principal do sistema.
- Ajustes de inicialização da aplicação.

### Commit #2: Funcionalidade Salvar implementada
- Implementação da lógica de salvar novos itens de leilão.
- Integração com a camada de persistência.

### Commit #3: Mensagem de sucesso/erro no cadastro
- Exibição de mensagens de feedback ao usuário após salvar.
- Mensagens diferenciadas para sucesso e erro.

### Commit #4: Listagem de itens funcionando
- Criação da tela de listagem de itens cadastrados.
- Exibição dinâmica dos dados salvos.

### Commit #5: Botão Editar implementado
- Inclusão da funcionalidade de edição de itens existentes.
- Ajustes na interface para permitir alteração dos dados.

### Commit #6: Botão Excluir implementado
- Implementação da funcionalidade de exclusão de itens.
- Confirmação antes de remover registros.

---

## 🚀 Tecnologias utilizadas
- Java
- NetBeans IDE
- Maven
- Swing (para interface gráfica)
